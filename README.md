## 介绍
本项目利用 GitHub Gist 作为数据库，搭建了一个基于Vue纯前端却可保存数据的移动端个人博客。

## 快速使用
点击右上方的登陆图标，绑定 GitHub 信息。

#### 用户名
用户名为 GitHub 的用户名。

#### Token
在 GitHub > Settings > Developer Settings > Personal access tokens 勾选 gist 权限获取 Token。
