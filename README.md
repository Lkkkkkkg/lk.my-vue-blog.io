## 介绍
本项目利用 GitHub Gist 作为数据库，搭建了一个纯前端可远程保存数据的单页面应用博客。
## 线上地址
https://lkkkkkkg.github.io/lk.my-vue-blog.io/
## 技术栈
vue <br>
vuex <br>
vue-router <br>
axios

## 快速使用
点击右上方的登陆图标，绑定 GitHub 信息。

#### 用户名
用户名为 GitHub 的用户名。

#### Token
在 GitHub > Settings > Developer Settings > Personal access tokens 勾选 gist 权限获取 Token。
